# Runtime Directory

This folder is reserved for ephemeral files, code scripts generated by agents during operation, temporary sandbox environments, and process logs.

## Future Architecture: Secure Sandboxing
- Agents will write generated code files to subdirectories inside `runtime/`.
- A sandbox runner (e.g., Docker container or WASM sandbox) can execute the scripts safely without risking the parent machine's security, and return execution stdout/stderr to the calling agent.
